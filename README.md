Convertor para Audio 2.0

Descrição breve do projeto.
Exemplo: Aplicação em Python com interface gráfica que converte texto em áudio.

Tecnologias
Python
customtkinter
gTTS
Instalação

Instale as dependências com:

pip install customtkinter gTTS
Como executar

Clone o repositório:

git clone https://github.com/seu-usuario/nome-do-repo.git

Acesse a pasta:

cd nome-do-repo

Execute o projeto:

python main.py
Observações
É necessário ter o Python instalado
O gTTS precisa de conexão com a internet
